# FieldHT

FieldHT is an iPhone app for controlling and configuring supported radios over Bluetooth.

Core product areas include:

- radio connection and status
- dual-channel radio control
- channel and memory group management
- programmable button configuration
- audio and accessory settings
- satellite tracking and live radio plan display

This repository contains FieldHT project assets used for App Store presentation and release support.
